# Variabel Random Univariat

Repository ini dibuat untuk memenuhi tugas mata kuliah Statistik mengenai
**Variabel Random Univariat**.

## Deskripsi
Variabel random univariat adalah variabel acak yang hanya melibatkan satu
variabel dalam suatu percobaan statistik. Repository ini berisi contoh dataset,
source code Python, dan visualisasi data univariat.

## Struktur Folder
- `dataset/` : berisi dataset univariat
- `src/` : berisi source code Python
- `README.md` : dokumentasi repository

## Dataset
Dataset yang digunakan merupakan data univariat dengan satu kolom nilai numerik.
Dataset ini digunakan untuk menghitung statistik dasar dan visualisasi distribusi data.

## Source Code
Program Python digunakan untuk:
- Membaca dataset CSV
- Menghitung mean, varians, dan standar deviasi
- Menampilkan histogram distribusi data
